PTERODACTYL PANEL BLUEPRINT & ADDONS INSTALLER
=================================================

🚀 Ultimate One‑Click Installer for Pterodactyl Blueprints & Addons  
Created by **7osteradev** — Fast, Clean, Fully Automated  
Compatible with **Ubuntu / Debian** systems and **Pterodactyl v1.x**

------------------------------------------------------------
📌 ABOUT THIS PROJECT
------------------------------------------------------------
This repository provides a powerful, modern, and fully‑automated script that
installs Blueprints and Addons for the Pterodactyl Panel in just ONE CLICK.

The installer includes:

✔ Automatic detection of all *.blueprint files  
✔ Interactive UI with beautiful animations  
✔ Safe installation system  
✔ Clean logs & error handling  
✔ Fully optimized for production servers  

Perfect for hosting providers, developers, and panel owners who want a fast,
reliable, and streamlined installation experience.

------------------------------------------------------------
🚀 ONE‑CLICK BLUEPRINT INSTALLER COMMAND
------------------------------------------------------------

Run this command to automatically install ALL blueprints:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/hopingboyz/Blueprint-installer/main/blueprint-installer.sh)
```

------------------------------------------------------------
🚀 ONE‑CLICK ADDONS INSTALLER COMMAND
------------------------------------------------------------

Run this command to automatically install addons:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/7osteradev/Blueprint-installer/main/addon-installer.sh)
```
